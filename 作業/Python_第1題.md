```
有四個數字：1、2、3、4，能組成多少個互不相同且無重複數字的三位數？各是多少？

分析：
可填在百位、十位、個位的數字都是1、2、3、4。組成所有的排列後再去掉不滿足條件的排列。
```

# 答案：
for a in range(1,5):  
  for b in range(1,5):
    for c in range(1,5):
      if ( a != b ) and ( a != c) and ( b != c):
        print (a,b,c)
